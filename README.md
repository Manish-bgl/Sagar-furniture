# Sagar Furniture Catalog

A modern, fast, and fully responsive digital catalog web application built for **Sagar Furniture**. 

### Key Features:
- **Client Catalog:** Clean category filters, live product search, premium product detail modals, and direct WhatsApp inquiry integration.
- **Admin Dashboard:** Secure Firebase Auth protected panel to add, edit, or delete designs with instant Cloudinary image upload.
- **Serverless Tech:** Real-time database sync via Firebase Firestore and 100% free hosting-ready architecture.
